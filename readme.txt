A decoupled React SPA for family note-taking and organization.

== Description ==

Family Notebook is a private, centralized notebook for your family. It functions as a single-page application (SPA) built with React, directly integrated into your WordPress environment.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/family-notebook` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Place the `[family_notebook_app]` shortcode on a page to render the app.
