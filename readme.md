# Family Notebook WordPress Plugin

A premium, app-like experience for families and community groups to organize notes, checklists, and shared information directly within WordPress.

## Features
* **Multi-Workspace Support**: Easily toggle between private family notebooks (e.g., "The Smith Family") and community workspaces (e.g., "Grace Church Group").
* **Cloaked User Experience**: Users register directly for the app without needing access to the WordPress backend.
* **Granular Role Management**: Includes **Owner**, **Organizer**, **User**, and **Viewer** roles to ensure privacy and security.
* **Contextual "Quick Add"**: A streamlined, mobile-friendly interface for creating notes instantly within folders.
* **Versatile Note Types**:
    * **Paragraph**: Rich-text editing for long-form notes.
    * **Interactive Checklist**: Perfect for grocery and packing lists with cross-out functionality.
    * **Outline**: Hierarchical bullet points for project planning.
    * **Grid/Tracker**: Spreadsheet-style format for budgeting or meal planning.
* **Customizable Branding**: Workspace-specific color themes for instant visual recognition.

## Architecture
* **Modern UI**: Fully responsive, mobile-first design with bottom-tab navigation.
* **Native Integration**: Leverages WordPress users and database structure, making it perfectly compatible with existing plugins like the Super Star Chore Chart.
* **Secure**: Built-in permission checks prevent unauthorized editing or access in community workspaces.

## Getting Started
1. Clone the repository into your `/wp-content/plugins/` directory.
2. Activate the plugin via the WordPress Admin dashboard.
3. Configure your Workspace settings and begin inviting members via Join Codes.
